# 14.1.剪绳子 (Medium)

## 题目描述

给你一根长度为 n 的绳子，请把绳子剪成整数长度的 m 段（m、n都是整数，n>1并且m>1），每段绳子的长度记为 k[0],k[1]...k[m] 。请问 k[0]\*k[1]\*...\*k[m] 可能的最大乘积是多少？

### 示例

> 输入: 2
> 
> 输出: 1
> 
> 解释: 2 = 1 + 1, 1 × 1 = 1

> 输入: 10
> 
> 输出: 36
> 
> 解释: 10 = 3 + 3 + 4, 3 × 3 × 4 = 36

### 提示

2 <= n <= 58

## 代码

数学问题，拆分要尽可能多的 3，如果最后余 1，那么应该拆成 2 + 2。

```c++
class Solution {
public:
    int cuttingRope(int n) {
        if(n <= 3) {
            return n - 1;
        }
        if(n == 4) {
            return 4;
        }
        unsigned int res = 1;
        while (n > 4){
            n -= 3;
            res *= 3;
        }

        res *= n;
        return  res;
    }
};
```

或者动态规划：

```c++
class Solution {
public:
    int cuttingRope(int n) {
        vector<int> dp(n + 1, 0);
        dp[1] = 1;
        for(int i = 2; i <= n; i++) {
            for(int j = 1; j < i; j++) {
                dp[i] = max(dp[i], max(j * (i - j), dp[j] * (i - j)));
            }
        }
        return dp[n];
    }
};
```
