# 69.x 的平方根 (Easy)

## 题目描述

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

### 示例

> 输入: 4
> 
> 输出: 2

> 输入: 8
> 
> 输出: 2

## 代码

简单的二分：

```c++
class Solution {
public:
    int mySqrt(int x) {
        if(x <= 1) {
            return x;
        }
        int l = 1, r = x;
        int mid = 0;
        int sqrt = 0;
        while(l <= r) {
            mid = l + (r - l) / 2;
            sqrt = x / mid;
            if(sqrt == mid) {
                return mid;
            }else if(mid > sqrt) {
                r = mid - 1;
            }else {
                l = mid + 1;
            }
        }
        return r;
    }
};
```

>
> 内存消耗：8.2MB
