# 876.链表的中间节点 (Easy)

## 题目描述

给定一个带有头结点 head 的非空单链表，返回链表的中间结点。

如果有两个中间结点，则返回第二个中间结点。

### 示例

> 输入：[1,2,3,4,5]
> 
> 输出：此列表中的结点 3 (序列化形式：[3,4,5])

> 输入：[1,2,3,4,5,6]
> 
> 输出：此列表中的结点 4 (序列化形式：[4,5,6])

## 代码

快慢指针。

```c++
class Solution {
public:
    ListNode* middleNode(ListNode* head) {
        // 跟回文那个差不多
        ListNode *slow = head, *fast = head;
        while(fast && fast->next) {
            fast = fast->next->next;
            slow = slow->next;
        }
        return slow;
    }
};
```
